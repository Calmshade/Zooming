A simple plugin to help you find the specific feature and zoom to it.


Features: Search for features by attribute, Automatically zoom to the selected feature, Highlights the feature even if zoom fails.

If zooming to the feature is not working, for some reason, you can find the feature selected in the attribute table.

Tip: Use this plugin when working with large datasets or overlapping layers where visual navigation is difficult.
